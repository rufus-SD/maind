# maind

**Your AI forgets everything when you close the chat. Maind fixes that.**

Python installer for [Maind](https://github.com/rufus-SD/maind) — a local, encrypted CLI that gives your AI tools persistent memory.

## Install

```bash
git clone https://github.com/rufus-SD/maind.git
cd maind
pip install -e python/
```

First run downloads the pre-built binary for your platform automatically.

## Usage

```bash
maind demo              # try it instantly
maind init              # set up your brain
maind connect cursor    # wire into your AI
maind                   # launch the dashboard
```

See the [full documentation](https://github.com/rufus-SD/maind) for details.
